# HostelLogic AI

HostelLogic AI is a modern, responsive web application designed to serve as a comprehensive administrative chatbot for hostel management. Acting as a friendly, intelligent, first-point-of-contact for students, it provides instantaneous, 24/7 conversational support regarding campus life, rules, and facilities.

This project pairs a robust Dialogflow ES agent backend with a beautifully animated, ultra-fast vanilla web interface.

## ✨ Features

- **Intelligent Virtual Assistance:** Engage with an advanced Dialogflow-powered chatbot capable of handling intent-based, multi-turn dialogs.
- **Seamless Student Onboarding:** Acts as a smart guide for new students, leading them step-by-step through various orientation and arrival procedures.
- **Rule Clarification & Info:** Quickly pulls up and clarifies regulations regarding hostel curfews, authorized visitors, and general code of conduct.
- **Mess & Dining Services:** Instantly queries current mess menus, daily dining schedules, and can accept student feedback or complaints directly.
- **Automated Issue Reporting:** Streamlines the maintenance workflow; students can report electrical, plumbing, network, or housekeeping issues directly within the chat window without long-form ticketing.
- **Premium User Interface:** Features a meticulously crafted Light-Theme interface utilizing structural glassmorphism, dynamic animations, fluid typing indicators, and sleek tailored gradients optimized for exceptional user engagement.

## 🛠️ Technology Stack

- **Frontend Architecture:** Clean HTML5 and Vanilla JS. Highly optimized, requiring zero heavy-weight client frameworks.
- **Dynamic Styling:** Written purely in Vanilla CSS incorporating CSS Variables for theme management, Flexbox for scaling, and custom `@keyframe` animations.
- **Backend / Integration:** Uses a Node.js-based Serverless API architecture to communicate securely with Google Cloud via the `@google-cloud/dialogflow` library.
- **Typography & Icons:** Polished with the modern `Inter` font family and integrated with lightweight `Lucide` icons.
